# Projeto Agronegócio – Sistema de Irrigação

Este projeto tem como objetivo analisar dados do agronegócio e desenvolver uma solução simples para controle de irrigação em diferentes culturas agrícolas.

## 📊 Análise de Dados (R)

Foi realizada uma análise exploratória utilizando a linguagem R, incluindo:

- Medidas de tendência central (média, mediana)
- Medidas de dispersão (variância e desvio padrão)
- Medidas separatrizes (quartis)
- Histograma da variável numérica
- Gráfico de barras para variável qualitativa

## 💻 Sistema em Python

Foi desenvolvido um mini sistema que permite:

- Cadastrar culturas agrícolas
- Registrar níveis de irrigação (Alto, Médio, Baixo)
- Listar os dados cadastrados
- Salvar os dados em arquivo JSON

## 🌱 Contexto do Problema

O projeto simula o controle de irrigação no agronegócio, permitindo organizar informações importantes para auxiliar na tomada de decisão.
